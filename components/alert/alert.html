Alert

Wing UI’s alert provides contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Basic Alert

Alerts are available for any length of text, as well as an optional close button. For proper styling, use one of the eight required contextual classes (e.g., .alert-danger).
